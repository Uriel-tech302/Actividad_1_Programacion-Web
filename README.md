# Actividad 1 - Programación Web

**Autor:** Uriel Espinoza de la Rosa

## 📋 Descripción
Este repositorio contiene los 18 ejercicios prácticos realizados durante la Actividad 1 de la materia de Programación Web. El proyecto tiene como finalidad organizar todos los trabajos en un solo sitio web, facilitando su consulta mediante una página principal (`index.html`) que funciona como menú de navegación.

## 🎯 Objetivos
*   Integrar los ejercicios prácticos en un repositorio público estructurado mediante carpetas.
*   Reforzar el uso de rutas relativas para enlazar correctamente páginas, estilos, scripts e imágenes.
*   Aplicar HTML5 para la estructura, CSS3 para el diseño visual y JavaScript para las validaciones e interacción.
*   Publicar el proyecto completo y funcional mediante GitHub Pages.

## 💻 Tecnologías Utilizadas
*   HTML5 & CSS3
*   JavaScript
*   Bootstrap (local)
*   SweetAlert (local)
*   XAMPP (Servidor local)
*   GitHub & GitHub Pages

## 🗂️ Resumen de Ejercicios
Este proyecto integra un total de 18 ejercicios prácticos desarrollados durante el curso, diseñados para avanzar progresivamente desde los conceptos básicos hasta la interactividad. Los temas principales que abarcan estas prácticas son:

*   **Estructura y Semántica HTML5:** Creación de documentos desde cero utilizando etiquetas semánticas (`<header>`, `<footer>`, `<section>`) para una correcta maquetación.
*   **Contenido y Multimedia:** Aplicación de formato a textos, organización de información mediante tablas y listas (ordenadas y desordenadas), e integración de recursos multimedia como imágenes.
*   **Navegación:** Implementación de hipervínculos internos, externos y anclas, comprendiendo el uso vital de las rutas relativas.
*   **Interactividad y Formularios:** Diseño de formularios de contacto funcionales con validaciones nativas de HTML5, y desarrollo de scripts básicos en JavaScript (como una calculadora conversora de temperatura) para procesar datos sin recargar la página.
*   **Entornos de Desarrollo:** Pruebas de funcionamiento en servidor local (XAMPP) y publicación final del entorno en línea.
